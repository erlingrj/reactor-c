◆ lf_readable_time()

size_t lf_readable_time ( char *  buffer,
instant_t  time 
)

Store into the specified buffer a string giving a human-readable rendition of the specified time. The buffer must have length at least equal to LF_TIME_BUFFER_LENGTH. The format is:

x weeks, x days, x hours, x minutes, x seconds, x unit

where each x is a string of numbers with commas inserted if needed every three numbers and unit is nanoseconds, microseconds, or milliseconds.

Parameters
bufferThe buffer into which to write the string.
timeThe time to write.
Returns
The number of characters written (not counting the null terminator).

◆ _lf_time_epoch_offset

interval_t _lf_time_epoch_offset
extern

Offset to _LF_CLOCK that would convert it to epoch time. This is applied to the physical clock to get a more meaningful and universal time.

For CLOCK_REALTIME, this offset is always zero. For CLOCK_MONOTONIC, it is the difference between those clocks at the start of the execution.

Offset to _LF_CLOCK that would convert it to epoch time. For CLOCK_REALTIME, this offset is always zero. For CLOCK_MONOTONIC, it is the difference between those clocks at the start of the execution.
