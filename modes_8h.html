◆ reactor_mode_t

typedef void reactor_mode_t

Header file of the runtime infrastructure for modes in the C target of Lingua Franca. This file contains user macros for setting new modes, forward declarations for functions and types used in the generated code, and types definitions for the modal representation.

Any mode related code will only work in the presence of the MODAL_REACTORS definition. However, this header should be included regardless to provide definitions for mode-unaware pre-compilation.

This file is intended for direct include in reactor.h augmenting certain type definitions.
